### Tic-Tac-Toe (Крестики-нолики)



Консольная игра «Крестики-нолики» на Python для двух игроков (PvP). 

Игра запускается из терминала и работает через ввод координат. 



###### Запуск:

Убедись, что установлен Python 3.x, затем в папке проекта выполни: 

python main.py 



###### Правила:

Играют два игрока: X и O 

Размер поля: 3×3 

Игроки ходят по очереди 

Для хода нужно ввести: 

номер\_строки номер\_столбца 

Пример: 

2 3 

Это означает: 2-я строка, 3-й столбец. 

&nbsp;

###### Условия победы 

Игрок побеждает, если соберёт 3 одинаковых символа подряд: 

по горизонтали 

по вертикали 

по диагонали 

Если поле заполнено и победителя нет — объявляется ничья.
